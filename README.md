# robosys2024

# 10パズル
![test](https://github.com/okowa42/robosys2024/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ4つの自然数を使って、結果が10になるような四則演算式を出力する。

## 使い方
- 任意の4つの自然数をスペース区切りで入力する。

## 必要なソフトウェア
- Python
    -テスト済みバージョン: 3.7～3.11

## テスト環境
- Ubuntu 24.04.1 LTS
